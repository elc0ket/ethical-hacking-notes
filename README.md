## Sobre este repositorio

Notas personales de estudio y práctica en hacking ético / pentesting, cubriendo desde fundamentos de redes y sistemas hasta especializaciones en AD, cloud, mobile, APIs y blockchain. El contenido combina teoría, comandos de referencia rápida y ejemplos prácticos usados en CTFs y laboratorios.
